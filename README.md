# Placement Survival Machine

Placement Survival Machine is a simple interview practice web app built using HTML, CSS, and JavaScript.

It helps users practice common placement interview questions by selecting a category and number of questions.

## Features

- Choose interview category
- Select number of questions
- Show interview questions
- Reveal answers
- Move to next question
- Mark questions as practiced
- Track number of practiced questions

## Categories Included

- HR
- JavaScript
- DSA
- OOP
- DBMS
- Operating System

## Tech Stack

- HTML
- CSS
- JavaScript

## Future Improvements

- Random question shuffle
- Better UI design
- Dark mode
- Progress bar
- Save progress using Local Storage
- More interview categories

## Project Goal

This project was built to improve JavaScript DOM manipulation, event handling, and interactive UI logic while creating something useful for placement preparation.
