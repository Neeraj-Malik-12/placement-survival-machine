# Placement Survival Machine

A simple and interactive interview preparation web app built using **HTML, CSS, and JavaScript**.

This project is designed to help users practice **interview questions** category-wise in a clean and engaging interface.

## Features (Current)
- Clean and responsive UI
- Category selection dropdown
- Question count selection
- Question display section
- Answer display section
- Action buttons:
  - Show Answer
  - Next Question
  - Mark Practiced
- Practice counter UI

## Planned Features
- Random question generation based on selected category
- Show / Hide answer functionality
- Next question navigation
- Mark as practiced functionality
- Practice count tracking
- localStorage support to save progress
